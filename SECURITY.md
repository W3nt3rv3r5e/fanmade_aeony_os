# Security Policy

## Supported Versions

AeonyOS: Zenith is a **fan-made conceptual project** — a static, browser-based desktop environment simulation built with vanilla HTML, CSS, and JavaScript. It is **not a real operating system** and does not run outside of a web browser.

Only the latest commit on the `main` branch is considered supported.

## Reporting a Vulnerability

We take security seriously even for this fun conceptual project.

If you discover a security vulnerability (for example, XSS issues, unsafe JavaScript behavior, or any concern that could affect users when running `index.html` in their browser), please report it responsibly.

**Please do not create public GitHub issues for security problems.**

Instead, report it privately by:

- Opening a **private vulnerability report** on GitHub (recommended):  
  Go to the **Security** tab → **Report a vulnerability**

- Or emailing the maintainer directly (if you prefer):  
  **w3nt3rv3r5e** at the domain associated with your GitHub account (or message the owner via GitHub).

Please include as much information as possible:
- Description of the vulnerability
- Steps to reproduce it
- Potential impact (e.g., affects user data, allows code execution in the browser, etc.)
- Any suggested fix or mitigation

## Disclosure Policy

- We will acknowledge receipt of your report within a few days.
- We will investigate and work on a fix as quickly as possible (this is a small hobby/fan project, so response time may vary).
- Once fixed, we will release an updated version and publicly thank you (unless you prefer to stay anonymous).

Because this is purely a **client-side browser simulation** with no backend, server, or persistent storage, most traditional security issues (remote code execution, server breaches, etc.) do not apply. The main risks are browser-related (XSS, clickjacking, unsafe dynamic content, etc.).

## Comments on this Policy

If you have suggestions to improve this Security Policy, feel free to open a pull request or issue labeled "security policy".
